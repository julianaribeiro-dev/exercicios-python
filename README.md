# exercicios-python

Repositório com exercícios propostos em sala de aula para estudos de algoritmo e lógica da programação utilizando Python.

Este repositório também é utilizado para praticar versionamento com Git e GitHub.

Os exercícios estão organizados em pastas por tema:

- `variaveis_e_calculos/`: Exercícios com entrada de dados e operações matemáticas.
- `if_elif_else/`: Exercícios com estruturas condicionais.
